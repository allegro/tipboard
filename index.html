Have you thought of presenting your business data, technical indices or team performance in a current and neat manner? Tipboard is a tool developed precisely for this purpose.

How to get started?

Design

Ready to use library of widgets allows you to present the data in an attractive way

Code

Quick tutorial explains how to feed the dashboard with the data

Enjoy

It is possible to contribute (github) and share your own widgets with the open source community

Tipboard enables you to:

  • define your own layout
  • use library of widgets and create your own widgets
  • easily retrieve data from sources like Jira (and Confluence - soon!)
  • feed widgets via REST API

Tipboard repository is published on Github. The project is licensed under the Apache License, Version 2.0. It's very easy to step up and help with developing future versions of Tipboard. Everybody can contribute by:

  • writing new widgets
  • improving existing widgets
  • writing connectors to data sources
  • testing
  • reporting bugs
